# Isle CPU

Isle uses a 32-bit RISC-V CPU.

_Building Isle_ chapters 6-8 use [FemtoRV32.v](FemtoRV32.v) ("electron", with RV32IM support) by Bruno Levy and Matthias Koch under [BSD-3-Clause license](FemtoRV32-LICENSE). Future chapters may use another RISC-V CPU, for example with interrupt support (TBC). See [From Blinker to RISC-V](https://github.com/BrunoLevy/learn-fpga/blob/master/FemtoRV/TUTORIALS/FROM_BLINKER_TO_RISCV/README.md) for the FemtoRV tutorial by Bruno Levy.

You can also find a RISC-V CPU design as part of [chapter 5](../book/ch05/).
