# Isle CPU

Isle uses a 32-bit RISC-V CPU, [FemtoRV32](FemtoRV32.v), supporting the RV32IMC instruction set.

The Isle edition of FemtoRV32 is derived from FemtoRV32 "Gracilis" by Bruno Levy and Matthias Koch under the [BSD-3-Clause license](FemtoRV32-LICENSE). You can find the original FemtoRV32 "Gracilis" in Bruno Levy's [learn-fpga](https://github.com/BrunoLevy/learn-fpga/blob/master/FemtoRV/RTL/PROCESSOR/femtorv32_gracilis.v), and the FemtoRV tutorial in [From Blinker to RISC-V](https://github.com/BrunoLevy/learn-fpga/blob/master/FemtoRV/TUTORIALS/FROM_BLINKER_TO_RISCV/README.md).
