# Isle Device Hardware

Devices present a standard MMIO interface to the the CPU. Devices typically wrap existing hardware modules.

| Module                           | Reference                    | Book Chapter    | Description                              |
| -------------------------------- | ---------------------------- | --------------- | ---------------------------------------- |
| [gfx_dev](gfx_dev.v)             | -                            | Input Output    | Graphics Device                          |
| [sys_dev](sys_dev.v)             | -                            | Input Output    | System Device (timers, lfsr, etc.)       |
| [uart_dev](uart_dev.v)           | -                            | Input Output    | UART Device                              |
