# Isle Projects

Projects that run on Isle.

## Graphics

[Starfield](starfield) is a simple animated starfield in hardware based on a [Linear-feedback shift register](https://en.wikipedia.org/wiki/Linear-feedback_shift_register). Build by editing `top_ch01.v` to use `starfield` in place of `ch01_square` then run ch01 make or build.tcl.
