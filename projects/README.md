# Isle Projects

These projects are based on a chapter of _Building Isle_.

## Chapter 1

Based on the hardware from [Chapter 1 - Isle Display](https://projectf.io/isle/display.html).

[Hitomezashi](hitomezashi) races the beam to create a stitch pattern, which appears both ordered and random. Build by editing `top_ch01.v` to use `hitomezashi` in place of `ch01` then run ch01 make or build.tcl.

[Starfield](starfield) is a simple animated starfield based on a [linear-feedback shift register](https://en.wikipedia.org/wiki/Linear-feedback_shift_register). Build by editing `top_ch01.v` to use `starfield` in place of `ch01` then run ch01 make or build.tcl. See [lfsr.v](../hardware/docs/lfsr.md) for Isle's LFSR Verilog module documentation.
